OOP Adventure Game

Project Description

For my second encounter with Javascript I made an adventure game inspired by Dragon Quest XI which used typed movement commands and displayed buttons when enemies or items were present. It was tested to ensure replayability since damage calculations mean no one attack is the same, currently working on a bigger better version now that I have learnt more on OOP and Javascript in general.

What I learned

What I learned from doing this project is how objects work within js and how you can make them interact with each other. I also got a better understanding on refactoring so I could reuse some code as a function rather than rewriting it over and over, I feel this is important because if there's something wrong with what you've written you only have to make changes to one code snippet rather than many.

Hobby/Interest Website Reworked

Project Description

I made a website for an interest on British tanks from World War 2. It incorporates responsive design so it is useable either on desktop and mobile, it also works well with darkmode browser addons.

What I learned

This is a rework on an earlier project where I didn't use any bootstrap, originally I wanted to take the first site and try incorporate bootstrap but eventually I figured it would be best (as well as being good practice) to remake it with bootstrap involved straight from the beginning. I learned how containers can align with each other and the different features that bootstrap provides. I learned how css stylesheets can be utilised in a powerful way to affect how things are displayed on the html pages.
